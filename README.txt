Bem-vindo ao Finance!

Para utilizar o programa, certifique-se de:

1. **Possuir sempre a Versão mais recente:**
   - Baixe a versão compilada mais recente do Finance

2. **Ter seu Python atualizado:**
   - Baixe sempre as atualizações do Python do site oficial "www.python.org"

3. **Tenha a biblioteca "requests instalada":**
   - Baixe utilizando o seguinte comando em seu terminal: "pip install requests"

**Em caso de não conseguir utilizar o executável siga os seguinstes passos:**

1. **Siga as instruções de bom uso do programa descritas acima**

2. **Clonar o Repositório:**
   - Clone este repositório em sua máquina local usando o comando:
     ```
     git clone https://github.com/bclepf/Finance.git
     ```
   - Ou baixe como um arquivo ZIP e extraia-o.

3. **Instale o pyinstaller:**
    - Baixe utilizando o seguinte comando em seu terminal: "pip install pyinstaller"

4. **Compile o programa em seu sistema:**
    - Utilize o seguinte comando "pyinstaller --onefile main.py"

5. **Localize e execute o arquivo gerado::**
    - Dentro da pasta de seu projeto localize a pasta "dist", nela deve conter o arquivo "main.exe", então o execute e aproveite a experiência, caso não o encontre verifique em qual pasta seu pyinstaller aloca os arquivos compilados.

Aproveite o Finance!
